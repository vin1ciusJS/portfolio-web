Sobre Mim

Me chamo Vinícius, sou formado na area de Hyper Text Markup Language, Cascading Style Sheets e JavaScript. Tenho 19 anos, atualmente estou cursando a faculdade de Sistemas da Informação, no qual pertenço ao 3° Semestre atualmente. Irei explicar o que é cada formação minha.

O HTML é uma ferramenta poderosa e essencial para qualquer desenvolvedor web. Ele permite que você estruture sua página da web, insira conteúdo como texto e imagens e crie links para outras páginas da web. É realmente a base de qualquer página da web que você vê na internet.

O CSS é uma ferramenta incrivelmente versátil que permite aos desenvolvedores web criar sites visualmente atraentes e responsivos.

O JS é uma linguagem de programação poderosa e flexível que é fundamental para a criação de experiências interativas na web
